# Customer Segmentation using K-Means Clustering

## Project Overview
This project segments customers based on Age, Annual Income, and Spending Score using the K-Means Clustering algorithm.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Synthetic customer dataset containing 200 customer records.

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Data Standardization
6. Elbow Method
7. K-Means Clustering
8. Cluster Visualization
9. Business Insights

## Output
The project groups customers into five clusters, helping businesses understand customer behavior and improve marketing strategies.
